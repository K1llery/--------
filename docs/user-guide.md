# User Guide

本文档面向演示和普通使用者，只描述页面操作；启动命令见 `README.md`。

## 登录

1. 打开前端首页。
2. 进入登录弹窗。
3. 使用演示账号登录；默认演示密码为 `demo123`。

## 主要页面

| 页面 | 职责 |
|---|---|
| 首页 | 展示系统入口、课程数据规模、地图入口和推荐证据入口。 |
| 数据洞察 | 查看 PPT 阈值完成度、热门排行、推荐 F1/Precision/Recall、分布和算法证据。 |
| 目的地 | 查看目的地列表、筛选标签、进入详情工作台、收藏和评分。 |
| 目的地详情 | 查看图片、浏览/评分统计、周边设施、美食、相关日记和路线入口。 |
| 搜索 | 演示精确、前缀和多关键词检索。 |
| 路线 | 通过问答式入口生成随便逛逛、指定地点到达和最近设施路线。 |
| 设施 | 按当前位置或指定场景查询附近设施。 |
| 美食 | 浏览校园和景区周边美食 POI。 |
| 日记 | 浏览、搜索、发布、评分、压缩/解压和生成 AIGC 分镜。 |

## 推荐演示顺序

1. 首页查看数据规模入口与精选目的地。
2. 数据洞察页查看 PPT 阈值、热门排行和推荐 F1 评估。
3. 目的地页筛选地点，进入目的地详情工作台并演示浏览计数、收藏和评分。
4. 搜索页输入校园或景区关键词。
5. 路线页选择“随便逛逛”，生成自动闭环路线。
6. 路线页选择“去指定地点”，切换步行、骑行或打车查看到达路线。
7. 路线页选择“找最近设施”，从图书馆出发查找最近公共厕所。
8. 设施页查看按图距离排序的结果。
9. 美食页按目的地坐标筛选周边 POI。
10. 日记页演示搜索、详情、浏览计数、登录评分、压缩/解压。
11. 日记详情页演示媒体画廊和 AI 分镜时间轴。
